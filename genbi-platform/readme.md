# MWS Data Compass GenBI

## Описание проекта

MWS Data Compass GenBI — концепция GenBI-платформы для получения аналитики через запросы на естественном языке.

Проект описывает AI-надстройку над существующей платформой MWS Data. Решение позволяет бизнес-пользователю задавать аналитический вопрос на русском языке и получать готовый результат: SQL-запрос, визуализацию или BI-дашборд.

Основная идея проекта: не заменять существующую data platform, а связать уже имеющиеся компоненты через новый orchestration-слой.

---

## Проблема

В текущем процессе бизнес зависит от аналитиков почти в каждом запросе к данным.

Основные ограничения:

*   бизнес-пользователь не знает SQL и структуру таблиц
*   аналитик остается обязательным звеном между бизнесом и данными
*   типовые ad-hoc задачи занимают значительную часть времени аналитических команд
*   time-to-insight измеряется днями, хотя бизнесу нужен ответ за минуты
*   компоненты платформы существуют отдельно и не связаны в единый пользовательский сценарий
