# Customer Churn Analysis

## Задача
Проанализировать отток клиентов и выявить ключевые факторы churn.

## Данные
Telco Customer Churn Dataset  
7043 клиентов, 21 признак

## Что сделано
- Разведочный анализ данных (EDA)
- Формирование гипотез
- Статистическая проверка (chi-square)
- Построение модели (Logistic Regression)

## Метрики модели
- Accuracy: 0.78
- Precision: 0.61
- Recall: 0.51
- ROC-AUC: 0.82

## Ключевые выводы
- Долгосрочные контракты снижают churn
- Новые клиенты чаще уходят
- Высокие платежи увеличивают риск
- Дополнительные услуги снижают отток

## Бизнес-ценность
- Можно выявлять клиентов с высоким риском
- Можно строить retention-кампании
- Можно снижать churn

## Стек
Python, Pandas, Scikit-learn, Matplotlib
